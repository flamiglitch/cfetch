# getmetal
getmetal prints basic information about the system that launches it (e.g., kernel, CPU).

## Build
1. download and cd into getmetal
```
$ git clone https://github.com/flamiglitch/getmetal.git
$ cd getmetal
```
2. compile
```
$ make
```
3. install
```
# make install
```
